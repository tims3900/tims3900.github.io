MBARI Summer Internship

Automated pipeline for kelp canopy coverage from RGB UAV imagery. Processed 26,000+ high-res images from Monterey Bay to monitor kelp forests without multispectral sensors.

Technical Details

Problem Inversion: Segment water background, then invert mask.

SAM + SAHI: Processed 42MP images by slicing into patches.

Seed Pointing: Automated placement using color + Poisson disk sampling.

Results

Processed 26,000+ images across 6 sites
Created ground truth dataset
